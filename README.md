# 🍽️ 레시피 주머니 - 나만의 비밀 레시피 공유 앱

나만이 알고 있는 비밀 레시피를 서로 공유하고 추천할 수 있는 애플리케이션


## 🧑‍💻 팀: 롤코 (RollCo)

| **이름** | **역할** |
|----------|----------|
| 최민석 | **팀장**, 서브 개발 (Android, Kotlin) |
| 김주환 | UI/UX 디자인 |
| 조윤재 | 데이터베이스 설계 및 구축 (Firebase) |
| 신현규 | 메인 개발 (Android, Kotlin) |


## ✨ 주요 기능 (무조건 구현할 기능)

*   **음성인식으로 조리 순서 이동**
*   **한 번에 두 가지 레시피 보기**
*   **API를 활용한 소셜 계정 로그인 기능**
*   **다양한 레시피 검색 및 추천**

## ⌛ 구현 예정 기능 (구현이 제한될 수 있는 기능)

*   **현재 날씨에 따른 맞춤 음식 추천**
*   **냉장고 재료에 맞춘 음식 추천**
*   **추후 추가 예정**
